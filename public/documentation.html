Terms & Conditions Summarizer

A lightweight browser extension that summarizes complex legal pages like Terms & Conditions and Privacy Policies into simple, easy-to-read bullet points ‚Äî powered by Gemini Flash 2.0 and optionally translated using Google Translate.

Installation

Chrome

  1. Open the Chrome Web Store.
  2. Search for "Terms & Conditions Summarizer".
  3. Click Add to Chrome.
  4. Pin the extension to your toolbar for quick access.

Microsoft Edge

  1. Go to the Edge Add-ons Store.
  2. Search for Terms & Conditions Summarizer.
  3. Click Get and confirm.
üí° Note: You don't need to restart your browser after installation.

Quick Start

Getting started with the Terms & Conditions Summarizer is simple:

  1. Visit a webpage that contains Terms & Conditions or similar legal text.
  2. Click the extension icon in your browser's toolbar.
  3. The extension will:
    • Extract the visible legal content from the page.
    • Generate a summarized version using Gemini Flash 2.0.
    • Highlight key clauses (e.g., auto-renewals, data-sharing).
    • Optionally translate the output to your selected language.
  4. View the results in a compact, friendly popup.
example_usage.dart
// The extension workflow follows this pattern
void main() {
  // Step 1: Visit a webpage with Terms & Conditions
  final webpage = WebPage('https://example.com/terms');
  
  // Step 2: Click the extension icon in browser toolbar
  final extension = TermsConditionsSummarizer();
  final result = extension.analyze(webpage);
  
  // Step 3: View the processed results
  print('Summary: ${result.summary}');
  print('Highlights: ${result.highlights}');
  print('Translation: ${result.translatedContent}');
}

How It Works

Content Extraction

The extension injects a script into the current tab to extract visible text from containers commonly used in legal documents (<article>, <div>, <main>). It filters unrelated content (navbars, footers, etc.) using a weight-based system.

content_extractor.dart
class ContentExtractor {
                String extract(Document document) {
                    // Inject script to extract visible text from common legal containers
                    final containers = document.querySelectorAll(['article', 'div', 'main']);
                    
                    // Filter unrelated content using weight-based system
                    return containers
                        .where((element) => _isLegalContent(element))
                        .map((element) => element.text)
                        .join('\n');
                }
                
                bool _isLegalContent(Element element) {
                    // Weight-based filtering algorithm
                    return element.textContent.containsAny([
                    'terms', 'conditions', 'privacy', 'policy'
                    ]);
                }
                }

Summarization Logic

Summarization is performed using Google Gemini Flash 2.0, optimized for fast responses and accurate contextual breakdowns. The output includes:

  • A general overview of the T&C
  • Risks (e.g., "you agree to arbitration")
  • Data and privacy implications
  • Cancellation/refund terms
summarizer.dart
class GeminiSummarizer {
                Future summarize(String content) async {
                    final gemini = GeminiFlash2();
                    
                    // Send content to Gemini Flash 2.0 API
                    final response = await gemini.summarize(
                    content: content,
                    includeSections: [
                        'overview',
                        'risks',
                        'data_privacy',
                        'cancellation_terms'
                    ],
                    );
                    
                    return Summary.fromResponse(response);
                }
                }

Translation (Optional)

If enabled, the summary is sent to the Google Translate API and returned in the user's preferred language.

translator.dart
class Translator {
  Future translate({
    required String text,
    required String targetLanguage,
  }) async {
    final translator = GoogleTranslateAPI();
    
    // Send summary to Google Translate API
    final translated = await translator.translate(
      text: text,
      target: targetLanguage,
    );
    
    return translated;
  }
}

APIs Used

Gemini Flash 2.0

  • POST endpoint for summarization
  • Input: Raw legal content
  • Output: Markdown or plain text summary
  • Auth: Bearer Token

Google Translate API

  • RESTful
  • Input: Target language + summary
  • Output: Translated summary
  • Auth: API Key

FAQ

Can I use this extension offline?

No. It needs internet access to communicate with the Gemini and Translate APIs.

Is my browsing data tracked?

No. The extension does not track, log, or store any browsing or usage data.

Enterprise support is being developed. Stay tuned for v2.0.
